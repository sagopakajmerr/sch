----belki lazım olur----

Converting Files from SCSS to Sass & Vice Versa

Convert Sass to SCSS
<code>
sass-convert styles.sass styles.scss
</code>

Convert SCSS to Sass
<code>
sass-convert styles.scss styles.sass
</code>
